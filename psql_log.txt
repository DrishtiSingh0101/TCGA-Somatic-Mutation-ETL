Data Analytics:

Phase 1: Descriptive Analytics
What does the dataset look like?
Question 1: How many mutations are present in each cancer?

SELECT cancer_type, COUNT(*) AS mutation_count FROM mutations GROUP BY cancer_type ORDER BY mutation_count DESC;

Question 2: What are the most frequently mutated genes?

SELECT gene, COUNT(*) AS g_mut_count FROM mutations GROUP BY gene ORDER BY g_mut_count desc LIMIT 20;

Question 3: What proportion of mutations are SNVs vs INDELs?

cancerdb=# SELECT MUTATION_TYPE, COUNT(*) AS COUNT FROM MUTATIONS GROUP BY MUTATION_TYPE;

Question 4: What proportion of mutations are SNVs vs INDELs in each cancer type?

SELECT CANCER_TYPE, MUTATION_TYPE, COUNT(*) AS COUNT FROM MUTATIONS  GROUP BY CANCER_TYPE, MUTATION_TYPE;

Question 5: Which mutation classifications are most common?

SELECT VARIANT_CLASSIFICATION, COUNT(*) AS COUNT FROM MUTATIONS GROUP BY VARIANT_CLASSIFICATION ORDER BY COUNT DESC;

Phase 2: Sample-Level Analytics

Question 5: What is mutation burden per sample?


SELECT SAMPLE_ID, COUNT(MUTATION_TYPE) AS MUT_COUNTS FROM MUTATIONS GROUP BY SAMPLE_ID ORDER BY MUT_COUNTS DESC;

MUTATION_TYPE COUNTS PER SAMPLE: 

SELECT SAMPLE_ID, MUTATION_TYPE, COUNT(MUTATION_TYPE) AS MUT_COUNTS FROM MUTATIONS GROUP BY SAMPLE_ID, MUTATION_TYPE;
or
SELECT SAMPLE_ID, MUTATION_TYPE, COUNT(MUTATION_TYPE) AS MUT_COUNTS FROM MUTATIONS GROUP BY SAMPLE_ID, MUTATION_TYPE ORDER BY MUT_COUNTS DESC;
